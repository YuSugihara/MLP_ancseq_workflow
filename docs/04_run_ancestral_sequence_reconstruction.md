# Step 04: Run ancestral sequence reconstruction

## Purpose

Run ancestral sequence reconstruction on the refined codon alignment with `ancseq`, using the selected outgroup sequence from the outgroup clade. The ML tree generated by `ancseq` is midpoint-rooted for downstream inspection.

## Input

- [data/two_mlp_clades/MLP_codon_clade.global_aln.fasta](../data/two_mlp_clades/MLP_codon_clade.global_aln.fasta)

## Command

```bash
scripts/04_run_ancseq.sh
```

The `ancseq` call is:

```bash
ancseq \
  -s data/two_mlp_clades/MLP_codon_clade.global_aln.fasta \
  -m DNA \
  --outgroup GCA_036507275.1_Cucurbitales_Sicyos_edulis_Helixer_CM071850.1_001929.1 \
  -o data/anceseq_results_v1.3.0
```
